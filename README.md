# RuStoreMVP

Приложение — MVP версия магазина приложений RuStore.

## Функциональность

- Онбординг при первом запуске
- Витрина приложений с карточками
- Список категорий приложений
- Карточка приложения с описанием и скриншотами

## Технологии

- Kotlin
- Jetpack Compose
- ViewModel
- Navigation
- Material3

- Для проверки необходимо распаковать и открыть архив в Android Studio
